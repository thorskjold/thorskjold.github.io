Apple's Xcode is an integrated programming environment (IDE) for Apple Developers,
and provides a Storyboard "life cycle" for Apple's UIKit (macOS) and AppKit (iOS) frameworks.
Storyboards allow developers to visually prototype and simultaneously generate application code.

Apple's more recent cross-platform SwiftUI framework doesn't provide a Storyboard interface,
and so Swiftboard is being developed to fill this gap for developers seeking to visually prototype
SwiftUI applications while simultaneously generating Swift application code.
Visualizing Statistical Analysis using Python
Bachelor-level University Course
Download ZIP
Python is a programming language with powerful data processing and analysis capabilities,
with a simple-to-learn syntax and accessible documentation. In this project, Python functions
were written and applied to a selection of datasets on the Danish population by Danmarks Statistik.

The goal was to output various graphs and visualizations, by identifying the structure of the datasets,
and utilizing the Python libraries Numpy, Pandas, and MatPlotLib. The code was designed with the aim
of modularity, meaning it was applicable to any datasets matching the structure of the selection.
