Arduino MKRZERO
for distance-based sensing
Ultrasonic sensing
Arduino provides a simple and affordable framework for constructing interactive experiences. With this Arduino setup, our project group sought to construct an invisible interface for interaction with an artistic installation. Utilizing an ultrasonic sensor with the Arduino MKRZERO, the installation played a sound clip through a connected speaker, determined by sensing the position of the user relative to the installation.
Hardware
Arduino MKRZERO, Samsung EB-P1100 Powerbank, MicroSD card SD-C02G, Breadboard & circuits, Ultrasonic distance sensor HC-SR04, Audio amplifier GF1002, Mono-audio speaker.
